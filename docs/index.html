The Tool

Reddit Skills Radar runs a three-stage pipeline: Rank (engagement × time decay) → Reduce Noise (language filters, de-duplication) → LLM Summaries (map-reduce that condenses posts into concise, defensible insights).

  • Trend score: catch emerging skills/tools
  • LLM map-reduce: mini-summaries → executive themes
  • Co-occurrence: see ecosystems form
  • JD Gap: paste a job description → get skill gaps

Why I Built It

People decisions need real-time signals. Surveys and vendor reports lag months; Reddit shows what practitioners wrestle with today. This app compresses the firehose into narratives leaders can trust, with LLMs generating summaries at scale.

Who it’s for

  • People Analytics & TA leaders aligning hiring to market skills
  • Learning/Enablement teams planning curricula
  • Strategy/Product teams tracking ecosystems
Tech Stack
  • Python · Streamlit · PRAW (Reddit API)
  • LLM map-reduce summaries (configurable backend)
  • Time-decay ranking, de-duplication, co-occurrence graph
  • Render (hosting)
